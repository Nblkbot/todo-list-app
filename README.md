# Todo List App

An Android application for managing your daily tasks.

## Features
- ✅ Add new todo items
- ✅ Mark items as complete
- ✅ Delete todo items
- ✅ Local data persistence
- ✅ Clean and intuitive UI

## Setup Instructions

### Prerequisites
- Android Studio (latest version)
- Android SDK 21 or higher
- Kotlin support enabled

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Nblkbot/todo-list-app.git
   cd todo-list-app
   ```

2. Open the project in Android Studio

3. Build and run on an emulator or physical device

## Project Structure
- `app/` - Main application code
- `app/src/main/java/com/example/todolist/` - Kotlin source files
- `app/src/main/res/` - Resources (layouts, strings, etc.)

## Usage
1. Launch the app
2. Enter a task in the input field
3. Tap "Add" to create a new todo
4. Tap the checkbox to mark as complete
5. Swipe or tap delete to remove items

## Technologies Used
- Kotlin
- Android Jetpack (ViewModel, LiveData)
- SQLite Database
- Material Design 3
